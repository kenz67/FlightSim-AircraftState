Introduction

This application was created to save the state of an aircraft in MSFS 2020 in order for it to be restored for the next flight allowing the user to begin where they left off. It will capture and reset the following data:

The data will be saved for each plane as defined by the plane Title from the aircraft.cfg file. The title as found will be displaed in the upper left of the main page.
  • Com 1 and 2, Active and Standby frequencies
  • Nav 1 and 2, Active and Standby frequencies
  • Adf Active and Standby frequencies
  • OBS 1 and 2 setting
  • ADR Card Setting
  • Fuel amount (gallons) in left and righ tanks
  • Fuel Selector switch setting
  • Trim Setting
  • Aircraft Location (Lat/Long)
  • Acraft Alititude
  • Acraft Heading
  • Kohlsman (Barometer) Setting
  • Heading Bug Indicator
  • Flaps configuration
  • Parking Break On/Off

Controls

This button will send the data from the previous flight to the simulator

This button will save the current data for later use

If connection to the Sim is lost, this will reconnect. Connection status can be seen in the status bar.

Settings

  • Show summary page before sending to sim
    • If checked, a summary of what will be sent to the sim will be shown before it is sent. This page gives the user the chance to override what is sent.
  • Auto save to db
    • If checked, the application will automatically save the data for the current flight to the database. The data will be saved when the master battery switch is turned off and settings have been sent to the sim.
  • Update Plane Location
    • If checked, the plane will be moved to the previous location. Note that this could end up with the plane sharing a parking spot with an AI generated plane.
  • Update Fuel
    • If checked, current fuel levels will be updated in the sim. This could cause problems with Career Mode, such as FS Economy which also want to set the fuel levels.

Send to Sim

The send to sim window will show the values the will be sent to the sim. There is also an option to override the default values for sending Location and Fuel to the sim.

Notes

I wrote this application for my own use as I like to continue my flights from one place to another with different planes on different routes as well as to learn the SimConnect interface. If time permits or I move into more complex aircraft I may add more data, but I don't have a lot of free time and prefer to spend it flying rather than coding. Even in its current limited state, I have found it very useful and thought others might as well. I have a home cockpit which has controls such as throttle quandrant, lights, fuel pump, etc. so none of this type of information is stored. The application has been tested with the Steam Version of the Cessna 172 and the plane Bonanza G36. It should work with any GA aircraft without issues. I do not fly airliners and no testing has been done in this area, but for the given data, it should work.

A bonus use I have found is after a Sim Crash. The application will contain the last configuration as of the crash allowing the user to click the Save to Db button. Once the sim is restarted, sending data to the sim will allow you to restart from the same place. If on an IFR flight plan, you usually have to request flight following, then you can reopen your flight plan.
